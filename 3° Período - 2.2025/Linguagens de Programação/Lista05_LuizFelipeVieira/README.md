Crie um programa que permita o gerenciamento de um arquivo CSV para armazenar dados de pessoas informados pelo usuário. Seu programa deve implementar:

a. uma função para solicitar que o usuário informe o nome, o número do telefone e o e-mail da pessoa e adicionar esses dados ao arquivo. 

b. uma função para exibir os dados de uma pessoa cujo nome será informado pelo usuário.

c. uma função para exibir todos os dados armazenados no arquivo.

d. um menu (de texto) para o usuário escolher qual funcionalidade deve ser executada.
